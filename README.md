# Анализатор ночных сбоев (Stream Log Analyzer)

Этот проект представляет собой высокопроизводительный скрипт на Python для анализа серверных логов в режиме реального времени. Скрипт обрабатывает лог-файлы в формате JSON, фильтрует ночные инциденты (с 00:00 до 04:00) и генерирует сводный отчет об ошибках.
Основная особенность проекта — использование концепции ленивых вычислений (Lazy Evaluations) и функционального программирования, что позволяет обрабатывать файлы логов гигабайтного объема с минимальным потреблением оперативной памяти.


 # Технологический стек

• Язык: Python 3.8+
• Библиотеки: 
    • itertools (функция filterfalse для ленивой фильтрации).
    • collections (класс Counter для мгновенного подсчета частоты ошибок).
    • datetime (для парсинга ISO-метрик и форматирования даты отчета).
    • json (построчный парсинг объектов).

# Структура проекта

• analyzer.py — основной скрипт анализатора.
• file_log_server.json — тестовый файл логов в формате JSON.
• README.md — документация проекта (этот файл).
